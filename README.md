# EndZone Edge

NFL prediction / betting-model web app. Users weight **Offense / Defense / Special Teams** (sliders summing to 100%) to reflect their football philosophy; the app predicts win % and point spread from real NFL stats.

Status: **planning + artifact recovery complete; build not yet started.**
